🎯 STEP 1: Install Java Development Kit (JDK)
─────────────────────────────────────────────

Your laptop needs Java to build Android apps.

Option A: Easiest (Recommended)
  1. Download: https://www.oracle.com/java/technologies/javase-downloads.html
  2. Look for: "Java SE Development Kit 11" (LTS version - most stable)
  3. Click "Download"
  4. Accept license, download "Windows x64 Installer"
  5. Run installer, accept defaults
  6. Click "Next" until done
  
  
Option B: Command line (OpenJDK)
  Windows Package Manager:
    winget install OpenLogic.OpenJDK.11
  
  Or download from: https://adoptopenjdk.net/ → OpenJDK 11

After installing, open NEW Command Prompt and test:
  java -version
  
Should print something like:
  java version "11.0.x" ...

🎯 STEP 2: Install Android SDK Build Tools
──────────────────────────────────────────

You already have: C:\Android\cmdline-tools with ADB

  2. Install it

🎯 STEP 3: Set Android SDK Path (ANDROID_HOME)
───────────────────────────────────────────────

After installing SDK, you need to add ANDROID_HOME environment variable:

1. Press: Windows key + "environment variables"
2. Click: "Edit the system environment variables"
3. Click: "Environment Variables" button
4. Click: "New" under User variables
5. 
   Variable name: ANDROID_HOME
   Variable value: C:\Android
   
6. Click: "New" again for PATH modification
   Variable name: PATH
   Variable value: C:\Android\tools\bin;C:\Android\platform-tools
   
7. Click OK, OK, OK
8. Close all terminals and open NEW Command Prompt

❓ QUESTIONS?

Q: Can I use Android Studio instead?
A: Yes! Android Studio comes with Java + SDK bundled.
   Download: https://developer.android.com/studio
   It's ~4GB but has everything you need.

Q: Why do I need Java?
A: Gradle (the Android build tool) is written in Java.
   Android build process requires JDK.

Q: My laptop is too slow for Android Studio...
A: Then just install JDK + minimal command-line tools.
   That's what we're doing - lighter weight!

Q: How much space do I need?
A: JDK: ~500 MB
   Android SDK: ~1-2 GB
   Total: ~3 GB
