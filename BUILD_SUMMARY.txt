   ├─ NavigationContainer wraps RootNavigator

   ├─ Top bar (title, notification badge, filter icon)
   ├─ Map view with incident markers
   ├─ Incident list below map
   └─ Floating action buttons (+ and 🔥)

3. User Actions (Select One):
   
   a) View Incident
      User taps incident card/marker
      ├─ IncidentDetailModal opens
      ├─ Shows type, distance, upvotes, description
      ├─ User can:
      │  ├─ Confirm (upvote)
      │  ├─ Share (coming soon)
      │  └─ Report (coming soon)
      └─ Close modal

   b) Report Incident
      User taps "+" button
      ├─ ReportModal opens
      ├─ Select type (4 options)
      ├─ Enter description
      ├─ Submit
      ├─ System creates incident
      ├─ Notifications sent to nearby users
      └─ Modal closes, incident appears on map

   c) Filter/Sort
      User taps "⚙️" button
      ├─ FilterPanel opens
      ├─ Select types (checkboxes)
      ├─ Choose radius (buttons)
      ├─ Choose sort (radio)
      ├─ Apply filters
      └─ Map updates with filtered incidents

   d) Check Notifications
      User taps "🔔" button
      ├─ NotificationsPanel opens (full-screen)
      ├─ View all notifications
      ├─ Tap notification to jump to incident
      ├─ Auto-marks as read
      └─ Badge count updates
