🐛 TROUBLESHOOTING:

Q: "JAVA_HOME is not set" error
A: The batch script should fix this. If not:
   - Check Java is installed: C:\Program Files\Java\jdk-26\bin\java.exe
   - Run batch script again

Q: "no devices found"
A: Before running batch script:
   - Connect phone via USB
   - Enable USB Debugging
   - Run: adb devices (should show your phone)

Q: Build takes very long
A: First build is slow (2-3 min). That's normal!
   Grab coffee ☕

Q: App crashes on launch
A: Check Terminal 1 (Metro) for errors
   Common issues: Missing module, syntax error
   Let me know the error message!

Q: "Module not found" error
A: npm dependencies might have issue. Try:
   cd e:\Projects\my-react-native-app
   npm install
