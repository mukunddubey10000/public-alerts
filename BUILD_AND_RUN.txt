🐛 TROUBLESHOOTING:

Q: After restart, "java: command not found"
A: Try setting PATH again:
   setx PATH "%PATH%;C:\Program Files\Java\jdk-26\bin"
   Then restart computer

Q: "npm start" fails
A: Make sure you're in: e:\Projects\my-react-native-app
   Run: cd e:\Projects\my-react-native-app
   Then: npm start

Q: "npm run android" says "no devices found"
A: Check phone connection:
   adb devices
   If not listed:
   - Disconnect USB cable
   - Wait 5 seconds
   - Reconnect cable
   - Check USB Debugging is ON

Q: Build fails with Gradle error
A: This is usually a first-time issue. Try again:
   npm run android

Q: App crashes on phone
A: Check Metro (Terminal 1) for errors
   Press "r" in Metro to reload
   Or restart both terminals
