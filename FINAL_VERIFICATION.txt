🔧 TROUBLESHOOTING:

If app won't start:
1. Check Metro is running (npm start in Terminal 1)
2. Try: npm run android -- --no-packager (use existing Metro)
3. Clear cache: npm start -- --reset-cache

If you see errors:
1. Read error message carefully
2. Check emulator is running
3. Try running again (sometimes it just needs retry)

If dependencies seem wrong:
1. They were just verified and are correct
2. Just run npm start and npm run android
