🎯 OPTION B: LIGHTWEIGHT (Minimal setup - for slow laptop)
──────────────────────────────────────────────────────────

You already have cmdline-tools. Now:

STEP 1: Install Java
  Option B1 - Easy:
    Go to: https://adoptopenjdk.net/
    Download: OpenJDK 11 (Windows x64)
    Run installer, click "Next" until done
  
  Option B2 - Command line:
    Open Command Prompt and run:
      winget install OpenLogic.OpenJDK.11

STEP 2: Use sdkmanager to download platforms
  
  After Java is installed, open NEW Command Prompt:
  
    C:\Android\cmdline-tools\bin\sdkmanager --list
  
  You should see list of packages available.
  
  Download what you need:
    C:\Android\cmdline-tools\bin\sdkmanager "platform-tools" "build-tools;30.0.3" "platforms;android-31"

  Wait 5-10 minutes for download.

STEP 3: Set ANDROID_HOME
  
  Press: Windows key + type "environment"
  Click: "Edit the system environment variables"
  Click: "Environment Variables"
  Click: "New"
  
  Variable name: ANDROID_HOME
  Variable value: C:\Android
  
  Click OK, OK, OK
  
  Close ALL terminals and open NEW Command Prompt

STEP 4: Try building
  
  cd e:\Projects\my-react-native-app
  npm start
  
  (In another terminal)
  npm run android

TIME: 20 minutes

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A: Make sure Java installed first and terminals are closed/reopened

Q: "ANDROID_HOME not set"
A: Did you close/reopen terminals after setting it?

Q: Download taking too long
A: It's normal, could be 10+ minutes for first download
